# Leilões Java

## 📌 Descrição
Sistema desktop em Java para cadastro e listagem de produtos em leilão.

## 💻 Tecnologias Utilizadas
- Java (Swing)
- MySQL
- JDBC
- NetBeans
- Git e GitHub

## ✅ Funcionalidades
- Cadastro de produto
- Consulta de produtos disponíveis
- Marcar produto como vendido
